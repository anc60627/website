Which CSS3 property can be used to indicate the text that flows out of the containing element should be clipped or indicated with an ellipsis?

  • A. text-wrap
  • B. text-overflow
  • C. text-flow
  • D. text-clip

    • text-overflow The text-overflow property specifies how overflowed content that is not displayed should be signaled to the user. It can be clipped, display an ellipsis (...), or display a custom string. Both of the following properties are required for text-overflow: white-space: nowrap; overflow: hidden;
