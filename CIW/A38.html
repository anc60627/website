Which property of the multi-column layout defines the number of columns an element should spread across?

(A) column-span is a CSS property that defines how many columns an element should span across in a multi-column layout. It can be set to either 1 (default) or all to span across all columns.

(B) column-gap is a CSS property that sets the width of the gap between columns in a multi-column layout.

(C) column-rule-style is a CSS property that sets the style of the rule (line) between columns in a multi-column layout. It can be set to various styles such as solid, dashed, dotted, double, etc.

(D) column-width is a CSS property that sets the width of each column in a multi-column layout. It can be set to a specific length value or to auto to automatically calculate the width based on the available space.
